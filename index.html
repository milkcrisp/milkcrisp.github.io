自傳

　　我是蘇柏霖，畢業於國立臺灣師範大學科技應用與人力資源發展學系。大學四年，我學習了人力資源管理、資訊管理及數位學習，透過畢業專題製作Robotfly，逐漸發現自己對軟體工程領域比較有興趣，尤其是前後端的開發。畢業後，我選擇參加資展國際的全端工程師就業養成班，系統學習前後端技術，為軟體開發職涯奠定了基礎。

　　在養成班期間，我參與了分組專題「食譜魔方」，與團隊成員共同開發出以食材管理為核心的智慧食譜推薦系統。此專案使用 ASP.NET Core MVC 進行後端資料管理，並採用 Vue.js 實現互動性強的前端介面。在開發過程中，我深入學習了 API 設計、資料庫操作及前後端整合，並體會到資料動態展示和組件化管理的重要性。這段經歷不僅讓我掌握了前後端整合技術，還加強了我的團隊協作和問題解決能力。

　　此外，我獨立完成了兩個個人專案：「BingoBingo 賓果賓果」和「丹丹漢堡點餐銷售系統」。這些專案進一步強化了我在 Windows Forms、C# 及 SQL 數據處理方面的技能，並提升了我對系統設計與應用的理解。

　　我對全端開發充滿熱情，希望將我在 Vue.js、C#、SQL Server、ASP.NET Core MVC 等方面的專長應用於貴公司所面臨的技術挑戰，特別是在系統整合和前後端開發的過程中。我渴望參與更具挑戰性的專案，並與貴公司共同成長。未來，我希望成為一名具備全方位技能的全端開發工程師，並隨著經驗的累積逐步成為技術領導者。我相信，在貴公司這樣充滿創新與挑戰的環境中，我能不斷提升專業能力，為公司的技術發展帶來實質貢獻。
